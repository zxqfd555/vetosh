# Onboarding Guide

Getting a new office online takes about 15 minutes:

1. Unbox the machine and connect it to power and water.
2. Scan the QR code on the drip tray to pair it with your Lumina Cloud
   workspace.
3. Assign the machine to a floor and set brewing profiles in the console.
4. Run the self-test brew; the machine calibrates grind size automatically.

For fleets above 50 machines our field team performs installation on site,
included in the Enterprise tier.
